# Bank QR Automation (Postman)
## Project Overview
This project is a part of the Bank QR Automation System developed using Postman.
The main goal of this automation is to:
 - Automatically generate random customers
 - Create tokens for each customer
- Assign services dynamically
- Simulate real-life user traffic
- Run bulk executions using Collection Runner
